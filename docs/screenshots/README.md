# Screenshot Guide

Add GitHub-ready screenshots to this directory using these filenames:

```text
gameplay.png
stats-drawer.png
leaderboard.png
```

Recommended captures:

- `gameplay.png`: an active round with player, creature markers, chase route,
  Targets panel, and compact HUD visible.
- `stats-drawer.png`: the open History tab with a selected persisted session
  and its catches.
- `leaderboard.png`: completed sessions ranked in the Leaderboard tab.

Use a consistent desktop viewport, avoid exposing unrelated desktop content,
and capture populated states rather than empty panels. After adding the files,
embed them in the root README under **Screenshots and Demo**.
